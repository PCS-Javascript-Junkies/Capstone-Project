About Choose Your Own Portland Adventure

"Choose Your Own Portland Adventure" was developed as a capstone project for the Portland Code School's Full Stack Javascript Program. Our goal was to integrate our newfound skills into a fun and interactive web applicaiton that was not only challanging to develop but represented the culmination of our work thus far.

Libraries and Frameworks Used:

  • Framework - Backbone
  • Server - Node.js and Express
  • Core Language - JavaScript
  • APIs - Yelp, Forecast.io, Google Maps
